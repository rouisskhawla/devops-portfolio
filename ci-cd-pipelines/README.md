# CI/CD & Automation Projects

This folder contains projects demonstrating **continuous integration and continuous deployment** pipelines for Spring Boot applications.  
Each project showcases different tools and approaches for automating builds, tests, code quality, containerization, and deployment to staging and production environments.

---

## Projects

| Project | Tool / Focus | Description | Link |
|---------|--------------|------------|------|
| DevOps CI/CD Pipeline | Jenkins | Complete CI/CD pipeline with Maven, SonarQube, Nexus, Docker, and deployment to staging VM | [View Repo](https://github.com/rouisskhawla/devops-ci-cd-pipeline-jenkins) |
| CI/CD Pipeline | GitHub Actions | Automated workflow with versioning, SonarQube, Docker, staging & production deployment | [View Repo](https://github.com/rouisskhawla/devops-ci-cd-pipeline-github-actions) |
| CI/CD Pipeline | GitLab CI | GitLab CI/CD with Docker & Shell runners, SonarQube analysis, and staging deployment | [View Repo](https://github.com/rouisskhawla/devops-ci-cd-pipeline-gitlab-ci) |

## Key Skills Demonstrated

- Pipeline orchestration with Jenkins, GitHub Actions, and GitLab CI  
- Automated testing, code quality analysis, and packaging  
- Containerization with Docker  
- Multi-environment deployment (Staging / Production)  
- Secure credential and secrets management
