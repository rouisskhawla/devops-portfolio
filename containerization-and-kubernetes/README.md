# Containerization & Kubernetes Projects

This folder contains projects demonstrating **Docker-based microservices, Kubernetes deployments, and Helm orchestration**.  
Each project highlights containerization best practices, CI/CD integration, and microservices management.

---

## Projects

| Project | Focus | Description | Link |
|---------|-------|------------|------|
| Microservices DevOps Monorepo | Docker, CI/CD, Monorepo | Consolidates backend, frontend, and CI/CD workflows in a single repository; each service has its own Dockerfile and pipeline | [View Repo](https://github.com/rouisskhawla/microservices-devops-monorepo) |
| Docker Microservices (Organization) | Docker Swarm, CI/CD | Multi-repo microservices with service discovery (Eureka), API Gateway, and Angular frontend; pipelines run on org-level self-hosted runner | [View Org](https://github.com/docker-microservices) |
| Kubernetes Helm Deployment | Kubernetes, Helm, CI/CD | Monorepo for microservice application with Jenkins pipelines and Helm-based deployments; standardized deployment and semantic versioning | [View Repo](https://github.com/rouisskhawla/kubernetes-helm-deployment) |
| Multi-Environment CI/CD: Minikube & Kubernetes Cluster | Kubernetes, Jenkins, Docker | CI/CD pipeline automatically deploying to Minikube (dev) or Kubernetes cluster (prod) based on webhook events; Helm manifests for environment-specific deployment | [View Repo](https://github.com/rouisskhawla/k8s-minikube-vs-cluster) |

## Key Skills Demonstrated

- Containerization with Docker & Docker Compose  
- Microservices architecture and monorepo setup  
- Kubernetes deployments using Helm charts  
- CI/CD integration with Jenkins and GitHub Actions  
- Environment-aware deployments (Minikube vs Prod Cluster)  
- Service discovery, orchestration, and secure local access
