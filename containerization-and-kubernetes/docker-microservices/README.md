# Docker Microservices (Organization)

**Skills:** Docker Swarm, CI/CD, Spring Boot, Angular, Service Discovery

**Overview:**  
Multi-repository setup implementing microservices with Docker Swarm, Eureka for service discovery, API Gateway, and Angular frontend.  
Automated CI/CD pipelines run on a self-hosted GitHub runner at the organization level.

**Key Highlights:**  
- Multi-repo microservices architecture  
- Docker Swarm across multiple VMs with overlay network  
- Service discovery with Eureka  
- CI/CD pipelines via GitHub Actions  
- Swagger API documentation per service  
- Secure local access using Nginx, HTTPS, and custom DNS
