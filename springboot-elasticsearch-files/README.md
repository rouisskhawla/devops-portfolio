# Spring Boot Elasticsearch

This project focused on **file storage and search using Elasticsearch**.  It demonstrate how to securely upload, index, and retrieve files from Elasticsearch using Spring Boot.

---

## Project

| Project | Focus | Description | Link |
|---------|-------|------------|------|
| Elasticsearch File Upload Service | File Storage & Search | Spring Boot application for uploading TXT, JSON, and XML files, automatically indexing them in Elasticsearch with secure HTTPS connections and dynamic indices | [View Repo](https://github.com/rouisskhawla/springboot-elasticsearch-files) |

---

## Key Skills Demonstrated

- Spring Boot REST API development  
- File type detection and dynamic indexing  
- Elasticsearch integration with TLS/SSL  
- Unit testing with JUnit and Mockito  
- Docker containerization for deployment  
- CI/CD automation with GitHub Actions  
- Secure storage and retrieval of files  

Explore the project repository for **implementation details, API usage, and CI pipeline setup**.
