# Monitoring & Logging Projects

This folder contains projects focused on **system observability**, including monitoring, logging, and performance testing.  
They demonstrate how to collect metrics, analyze logs, and visualize system behavior in real time.

---

## Projects

| Project | Focus | Description | Link |
|---------|-------|------------|------|
| Monitoring with Grafana, Prometheus & k6 | Monitoring & Load Testing | Real-time metrics collection with Prometheus, visualization in Grafana, and performance testing using k6 | [View Repo](https://github.com/rouisskhawla/monitoring-grafana-prometheus-k6) |
| ELK Monitoring Stack | Centralized Logging | Spring Boot application integrated with Elasticsearch, Logstash, and Kibana for centralized logging and visualization | [View Repo](https://github.com/rouisskhawla/elk-monitoring-stack) |

---

## Key Skills Demonstrated

- Metrics collection with Prometheus  
- Dashboard creation and visualization with Grafana  
- Load and performance testing with k6  
- Centralized logging with ELK Stack (Elasticsearch, Logstash, Kibana)  
- Log analysis and troubleshooting  
- Observability and system performance monitoring  

Explore each project folder for **implementation details, dashboards, and logs**.
