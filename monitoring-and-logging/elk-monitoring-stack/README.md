# ELK Monitoring Stack

**Skills:** Elasticsearch, Logstash, Kibana, Spring Boot, Logback, TLS/SSL

**Overview:**  
Spring Boot application integrated with the ELK stack to centralize, process, and visualize logs in real time across different environments.

**Key Highlights:**  
- Centralized logging using Elasticsearch, Logstash, and Kibana  
- Environment-aware logging (dev vs production)  
- Structured logs with Logback configuration  
- Log ingestion via Logstash TCP pipeline  
- Secure Elasticsearch setup with SSL and authentication  
- Real-time log visualization using Kibana dashboards
